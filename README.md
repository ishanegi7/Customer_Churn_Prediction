## 📖 Project Overview

Customer churn prediction is a Machine Learning project that identifies customers who are likely to discontinue a service. By analyzing customer demographics, account information, and service usage patterns, the model helps businesses take proactive measures to improve customer retention.

This project was developed using Python in Google Colab and follows a complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), model training, and performance evaluation.

* Perform data cleaning and preprocessing
